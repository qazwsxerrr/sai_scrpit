# seer-bot

Automation skeleton for https://s.61.com/home/ daily web game tasks.

## Phase 0 quickstart

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Install Playwright browsers:

```bash
playwright install chromium
```

Optional setup script:

```bash
scripts/setup.sh
```

Run CLI help:

```bash
python -m bot.cli --help
```
